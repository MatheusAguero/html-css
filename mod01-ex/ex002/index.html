Imagens

Abrir imagem no html: "img"
Aparecerá: <img src="" alt=""> src -> source ou origem alt -> alternativo(alternativo se não aparecer a imagem)
Site com imagens sem direitos autorais: Unsplash; Pexels;

Formatos para imagem WEB

Principais :
JPG(1983) - Compactação de imagens; PNG(1996) - Permite imagem sem fundo

Outros:
GIF(1989)- Permite transparência e animação.

Tamanho maximo necessario de imagem para um site: 1500 pixel
